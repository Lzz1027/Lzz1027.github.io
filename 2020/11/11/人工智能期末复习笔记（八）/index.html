Markov processes 马尔科夫过程

我们有时间段 t = 0，1，2，…

在每个周期t中，世界处于某一状态St

马尔可夫假设：给定St， St+1 独立于所有 i < t 的 Si

​ P(St+1 | S1 , S2 , …, St ) = P(St+1 | St )

​ 给定当下的状态，历史没有告诉我们更多关于未来的东西

通常情况下，所有CPTs都是一样的:

对于所有的：P(St+1 = j | St = i) = aij (stationarity assumption 平稳假设)

天气例子

St is one of {s, c, r} (sun, cloudy, rain)

转换概率：

还需要指定初始分布P(S0)

始终，假设P(S0 = s) = 1

两天后下雨的概率是多少?

P(S2 = r)

P(S2 = r) = P(S2 = r, S1 = r) + P(S2 = r, S1 = s) + P(S2 = r, S1 = c) = 0.1*0.3 + 0.6*0.1 + 0.3*0.3 = 0.180

三天后下雨的概率是多少?

计算效率的方法：

​ P(S3 = r) = P(S3 = r, S2 = r, S1 = r) + P(S3 = r, S2 = r, S1 = s) + …

从n个周期到未来，需要对3的n-1次方条路径求和

更有效的方法

P(S3 = r) = P(S3 = r, S2 = r) + P(S3 = r, S2 = s) + P(S3 = r, S2 = c)

​ = P(S3 = r | S2 = r)P(S2 = r) + P(S3 = r | S2 = s)P(S2 = s) + P(S3 = r | S2 = c)P(S2 = c)

困难的地方是计算P(S2)

主要思路，计算分布P(S1)，然后P(S2)，然后P(S3)

周期数是线性的!

平稳分布

当t趋于无穷时，“一般”，分布P(St)将收敛于一个平稳分布

由概率πi给出的分布(其中i为状态)是平稳的，如果:

​ P(St = i) =πi 意味着 P(St+1 = i) =πi

当然,

P(St+1 = i) = Σj P(St+1 = i, St = j) = Σj P(St = j) aji

平稳分布是由πi = Σj πj aji定义

计算平稳分布

πs = 0.6πs + 0.4πc + 0.2πr

πc = 0.3πs + 0.3πc + 0.5πr

πr = 0.1πs + 0.3πc + 0.3πr

Hidden Markov models (HMMs) 隐马尔科夫模型

和马尔科夫模型类似，但我们看不到状态

相反地，我们只能看到每个阶段的一个 observation (观测)，这个观测依赖于当前状态

仍然需要一个转换模型：P(St+1 = j | St = i) = aij

也需要一个观测模型： P(Ot = k | St = i) = bik

拓展到HMM的天气案例

转换概率：

b.sw = 0.1 晴天湿的概率是0.1

b.cw = 0.3 多云湿的概率是0.3

b.rw = 0.8 下雨湿的概率是0.8

一般情况下：解决 P(S.t , O.0 = o.0 , O.1 = o.1 , …, Ot = o.t ) ，t=1,2，….

事后问题 ？？？？？？？？？

需要求得：P(S1 = r | O0 = d, O1 = w, O2 = w, O3 = d)

贝叶斯规则的“局部”应用:

P(S1 = r | O0 = d, O1 = w, O2 = w, O3 = d)

= P(S1 = r, O2 = w, O3 = d | O0 = d, O1 = w) / P(O2 = w, O3 = d | O0 = d, O1 = w)

所以真正要求得的是：P(S1 , O2 = w, O3 = d | O0 = d, O1 = w)

P(S1 = r, O2 = w, O3 = d | O0 = d, O1 = w)

= P(S1 = r | O0 = d, O1 = w) P(O2 = w, O3 = d | S1 = r)
