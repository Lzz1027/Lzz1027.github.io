Chapter 14 Bayesian networks：

为什么联合概率分布很重要

联合概率分布给出P(X1…Xn)

可以计算任意事件的概率，然后用于:

​ 条件概率：P (X1 | X2…Xn)

​ 同时发生概率：P (X1，X2)

指定概率分布

为每个原子事件指定一个概率是不切实际的

对于X1，…，Xn的值的每一个组合X1，…，Xn，都需要指定P(X1，…，Xn)

​ 如果每个变量有k个可能的值

​ 我们需要指定k的n次方 - 1个概率!

我们已经看到，使用(条件)独立性可以更容易地指定概率分布

贝叶斯网络允许我们

​ 指定任何分布，

​ 如果存在(条件)独立性，以自然的方式简明地指定这种分布

条件独立

我们说两个变量，A和B，在给定C时是条件独立的，如果:

​ P(A|BC) = P(A|C)

​ P(AB|C) = P(A|C)P(B|C)

这有什么帮助?

对于每个给定父变量的变量，我们只存储一个conditional probability table (CPT) 条件概率表(CPT)

什么是贝叶斯网络

一个有向无环图directed acyclic graph（DAG）

节点独立于与其不是父子关系的节点

Rigged casino 被操作的赌场的案例

说明概率分布的一般方法

变量是X1，…，Xn

P (X1,…,Xn) = P (X1) P (X2 | X1) P (X3 | X1, X2)……(Xn-1 Xn | X1,…)

或者:

P (X1,…,Xn) = P (Xn) P (Xn-1 | Xn) P (Xn-2 | Xn, Xn-1)…P (X1 | Xn…,X2)

可以指定每个组件

对于|右边变量的每个值组合，指定概率除以左边变量的值

如果每个变量都可以取k个值，

P(Xi|X1，…，Xi-1)需要(k-1) k-1值

Σi={1,..,n}(k-1)ki-1 = Σi={1,..,n}k i -k i-1 = k n - 1

与指定所有原子事件的概率相同——当然，因为我们可以指定任何分布!

图形化表示的影响

条件独立 救援!

问题：P(Xi|X1，…，Xi-1)要求我们指定太多的值

假设X1，…，Xi-1划分成两个子集，S和T，因此在给定S的情况下，Xi是条件独立于T的

P(Xi|X1，…，Xi-1) = P(Xi|S, T) = P(Xi|S)

只需要 (k-1)k的|S|次方个 值，而不需要(k1)k的i-1次方个值

图形化表示的影响

如果给定X1和X3, X4有条件地独立于X2

雨和洒水器的例子

洒水器与雨水无关，所以它们之间没有边

操纵赌场的例子

给定的die 2有条件地独立于die 1

赌场操纵，所以他们之间没有边

更复杂的雨和洒水器的例子

能很容易的计算出每个原子事件的概率

P(+r,+s,+n,+g,+d) = P(+r)P(+s)P(+n|+r)P(+g|+r,+s)P(+d|+n,+g)

还可以轻松地对原子事件进行示例

要想知道P(+r|+d) = P(+r, +d) / P(+d)

P(+r,+d)= ΣsΣgΣn P(+r)P(s)P(n|+r)P(g|+r,s)P(+d|n,g) =

​ P(+r)ΣsP(s)ΣgP(g|+r,s)Σn P(n|+r)P(+d|n,g)

变量消去法

从因子 Σn P(n|+r)P(+d|n,g) 中，我们求和n得到一个仅依赖于g的因子

[Σn P(n|+r)P(+d|n,+g)] = P(+n|+r)P(+d|+n,+g) + P(-n|+r)P(+d|-n,+g) = 0.3 *0 .9 + 0.7 * 0.5 = 0.62

[Σn P(n|+r)P(+d|n,-g)] = P(+n|+r)P(+d|+n,-g) + P(-n|+r)P(+d|-n,-g) = 0.3 * 0.4 + 0.7 * 0.3 = 0.33

继续往左，g接下来要求和，等等。

消除顺序影响

P(+r,+d)= ΣnΣsΣg P(+r)P(s)P(n|+r)P(g|+r,s)P(+d|n,g) = P(+r)ΣnP(n|+r)ΣsP(s)ΣgP(g|+r,s)P(+d|n,g)

在这种情况下，最后一个因子取决于两个变量!

不必总是对所有变量求和

可以删除网络中不相关的部分吗

P(+r, +s) = P(+r)P(+s) = .6*.2 = .12

P(+n, +s) = ΣrP(r, +n, +s) = ΣrP(r)P(+n|r)P(+s) = P(+s)ΣrP(r)P(+n|r) = P(+s)(P(+r)P(+n|+r) + P(-r)P(+n|-r)) = .6*(.2*.3 + .8*.4) = .6*.38 = .228

P(+d | +n, +g, +s) = P(+d | +n, +g) = .9
