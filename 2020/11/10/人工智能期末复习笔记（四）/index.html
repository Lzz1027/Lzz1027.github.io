一个故事

你的室友回家：他/她完全湿了

你知道以下事情:

​ 你的室友湿了

​ 如果你的室友是湿的，那是因为下雨了，或者是因为洒水器，或者两者都有

​ 如果你的室友因为洒水器而湿了，那洒水器一定是开着的

​ 如果你的室友被雨淋湿了，你的室友一定没带伞

​ 雨伞不在伞架里

​ 如果伞不在伞架里，要么你一定带着伞，要么你的室友一定带着伞

​ 你没有带伞

你能断定洒水器是开着的吗?

人工智能能断定喷头开了吗?

根据这个故事可以获得的知识：

室友湿了

室友湿了 => 室友因为下雨湿了 OR 室友因为洒水器湿了

室友因为洒水器湿了 => 洒水器开着

室友因为下雨湿了 => 室友没带伞

伞不在

伞不在 => 你带了伞 OR 舍友带了伞

你没带伞

Syntax 语法

知识库中结构良好的句子是什么样子的?

BNF语法：

Symbol → P, Q, R，…，RoommateWet，…

Sentence → True|False|Symbol|Not(Sentence)|(Sentence AND Sentence)|(Sentence OR Sentence)|(Sentence => Sentence)

我们有时会省略括号，但在形式上，应该一直写括号

Semantics 语义

一个model(模型)指定哪些命题符号是真的，哪些是假的

给出一个模型，我应该能够告诉你一个句子是对的还是错的

Caveats 警告

2是偶数 OR 3是奇数 是真的(不排除或)

2是奇数 => 3是偶数 为真(如果左边为假，它总是为真)

所有这些都是假设这些符号被赋予了它们的自然值…

Tautologies 重言式

如果一个句子对它的命题符号的任何设置都是真的，那么它就是一个重言式

Logical equivalences 逻辑等价

如果两个句子在每一组命题变量中都有相同的真值，那么它们在逻辑上是等价的
