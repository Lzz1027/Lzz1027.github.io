Ant-Design-Vue - Checkbox 组件

1. 项目结构：

项目包含测试文件夹 _test_ ，样式文件夹 style ，和 index.ts、Group.tsx、Checkbox.tsx 三个文件。

其中index.ts为组件入口，Checkbox.tsx实现组件基本功能，Group.tsx实现选项组

2. index.ts

通过 Ant-Design-Vue 的 /components/index.ts 入口文件，本文件作为组件入口（components\checkbox\index.ts）被导入，其中导入了 Checkbox组件和CheckBoxGroup组件，

在导入时，组件被use函数使用，触发install，将组件及子组件注册，并导出为 Checkbox，Plugin 和 CheckboxGroup；

3. Checkbox.tsx

3.1. 执行顺序

这里的几个函数执行顺序如图所示，当 setup() 完成后，进入 render() 函数，在实例挂载完毕后，mounted() 函数开始执行，最后在实例卸载前执行 beforeUnmount()

3.2. props

props 里设置了如下属性以及一些其他属性和功能

参数 说明 类型 默认值
autoFocus 自动获取焦点 boolean false
checked 指定当前是否选中 boolean false
defaultChecked 初始是否选中 boolean false
disabled 失效状态 boolean false
indeterminate 设置 indeterminate 状态，只负责样式控制 boolean false

configProvider用来接收由 components\config-provider\index.tsx 中的 defaultConfigProvider 提供的默认设置；
