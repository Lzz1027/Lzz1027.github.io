2. 插件安装

2.1. 安装方式：

点击 VsCode 界面左侧 Tab 栏的第五个按钮（或快捷键 Ctrl + Shift + X）即可进入安装插件界面，在上方搜索栏搜索插件名即可搜索安装：

2.2. 常规插件：

Chinese Language Pack

中文语言包（由于语言冲突不建议使用）

VsCode-icons & Material Icon Theme

目录图标美化

Path Intellisense

自动提示文件路径

Change-case

修改命名规则（支持驼峰、下划线等）

Bracket Pair Colorizer

颜色匹配括号

vscode-faker

随机生成姓名、地址、图像、电话号、乱数假文段落等内容，帮助快速生成测试用例；

用法：Ctrl/Command Shift P 呼出控制台，输入faker即可看到提示

Git History

图表方式查看Git日志

GitLens

实时查看代码修改者及提交时间

HTML/CSS 插件：

Auto Close Tag

自动闭合 HTML/XML 标签

Auto Rename Tag

自动完成另一侧标签同步修改

HTML CSS Support

智能提示 CSS 类名和 ID

HTML Snippets

智能提示HTML标签及含义，包含HTML5标签

HTML Boilerplate

HTML模板插件，输入html出现提示框，即可立即生成html结构

CSS Peek

从Html标签直接定位到CSS类，方便修改样式

Color Info

提供CSS颜色信息，辅助预览颜色效果

JavaScript 插件：

JavaScript Snippets Pack

JS代码高亮、提示补全等

JavaScript(ES6) Code Snippets

ES6代码高亮、提示补全等

TypeScript 插件：

TypeScript Extension Pack

ts扩展包，包括TsLint、TypeScript Hero、Json2ts等8个常用插件

Vue 插件：

Vue

vue代码高亮、补全提示等

Vue 3 Snippets

Vue3代码高亮、补全提示等

Vite(预览版)

vue文件右键一键运行

Tailwind CSS 插件：

Tailwind CSS IntelliSence

TailwindCSS提供代码高亮

Ant Design 插件：

Ant Design Snippets

提供代码高亮、补全提示
