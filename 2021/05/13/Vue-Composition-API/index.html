1. 什么是组合式API

常规的 Vue 组件中，有 data、computed、methods、watch等组件选项，组件的各种数据和方法分散在不同位置，对于同一个逻辑关注点经常需要跳转，难以阅读和维护。

组合式 API 将同一个逻辑关注点相关代码配置在一起。

2. 组合式 API 基础

2.1. setup 组件选项

在创建组件之前，setup 组件先被执行，解析 props，作为组合式 API 的入口点。

由于执行 setup 时组件尚未创建，所以 setup 中没有 this，且除了 props 之外无法访问组件中声明的其他任何属性（本地状态、计算属性、方法）
